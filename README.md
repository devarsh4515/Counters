# Verilog HDL Counter Designs

This repository contains Verilog HDL implementations of different types of counters.

## Projects Included
- Up Counter
- Down Counter
- Ripple Up Counter

## Contents
Each project includes:
- Verilog source code (`.v`)
- Testbench (`*_tb.v`)
- Simulation script (`run.sh`)

## Tools Used
- Verilog HDL
- Verilator
- GTKWave
- Git & GitHub
